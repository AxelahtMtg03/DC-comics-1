# 🦸‍♂️ Justice League - DC Heroes

## 📌 Présentation

Justice League - DC Heroes est un site web statique développé en **HTML** et **Bootstrap**.

Il s'agit de mon **tout premier projet web**, réalisé en **début d'année 2021** dans le cadre de mon apprentissage du développement **front-end**.

Le site présente les membres emblématiques de la Justice League : **Batman, Superman, Wonder Woman, Flash, Aquaman et Green Lantern**.

---

## 🎯 Objectifs du projet

- ✔️ Découvrir et structurer une page web avec **HTML5**
- ✔️ Découvrir l'utilisation d'un framework CSS avec **Bootstrap**
- ✔️ Comprendre le système de **grille responsive**
- ✔️ Créer mon premier site web fonctionnel
- ✔️ Apprendre les bases du développement web

---

## 🚀 Fonctionnalités

- 🦇 Présentation des héros de la Justice League
- 🖼️ Mise en page avec cartes et images
- 📱 Design responsive grâce à Bootstrap
- 🎨 Mise en forme simple et épurée
- 🔗 Navigation entre les sections

---

## 🛠️ Technologies utilisées

- **HTML5**
- **Bootstrap 4/5** (framework CSS)
- Pas de JavaScript, pas de back-end

---

## 📖 Contenu du site
Le site présente les héros suivants :

-  Batman - Le chevalier noir de Gotham

-  Superman - L'homme d'acier

-  Wonder Woman - La princesse Amazone

-  Flash - L'homme le plus rapide du monde

-  Aquaman - Le roi des océans

-  Green Lantern - Le protecteur du secteur 2814

---

## Ce que ce projet m'a appris:

- Les bases du HTML (balises, structure, sémantique)

- L'utilisation d'un framework CSS pour accélérer le développement

- Le système de grille responsive de Bootstrap

- Les premiers pas dans le monde du développement web

- La satisfaction de créer mon premier site du début à la fin
